# ThaiNews.Ai 🇹🇭 戰情室

專為泰國市場情報設計的 Streamlit 新聞聚合與分析儀表板。

## ✨ 特色 (Features)
- **總體經濟**: 泰國政經局勢與台泰關係最新動態。
- **產業趨勢**: 聚焦 PCB、伺服器與電子製造供應鏈。
- **重點台商**: 追蹤 10 大指標台廠（台達電、鴻海、廣達等）。
- **行動優化**: 手機版面自動適配，閱讀更舒適。

## 🛠️ 技術棧 (Tech Stack)
- **Python**
- **Streamlit**: Web 應用框架
- **Feedparser**: RSS 新聞爬蟲

## 🚀 快速開始 (Usage)

1. **安裝依賴**:
```bash
pip install -r requirements.txt
```

2. **啟動應用**:
```bash
streamlit run app.py
```
